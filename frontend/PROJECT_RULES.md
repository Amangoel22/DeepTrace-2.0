# DeepTrace Project Rules

## Project Overview

DeepTrace is an AI-powered media verification platform focused on deepfake detection.

The platform allows users to:

* Upload videos
* Analyze deepfake probability
* View heatmaps
* View suspicious timestamps
* Understand AI-generated reasons
* Download reports
* Store analysis history

Future Features:

* Audio deepfake detection
* Cross-modal verification
* Browser extension

---

## Tech Stack

* React
* Vite
* Tailwind CSS
* React Router
* Framer Motion
* React Icons

---

## Design Philosophy

The product should feel like:

* Modern SaaS
* Cybersecurity platform
* Digital forensics tool

Inspirations:

* Vercel (hero sections)
* Linear (cards and typography)
* Professional cybersecurity dashboards

---

## Styling Rules

Use Tailwind CSS only.

Do NOT use:

* Bootstrap
* Material UI
* Chakra UI
* Ant Design

Use reusable components.

---

## Folder Structure

src/

* components/

  * layout/
  * sections/

* pages/

* context/

* data/

---

## Component Naming

Use PascalCase.

Examples:

* Navbar.jsx
* Hero.jsx
* DashboardPreview.jsx

---

## Theme

Support:

* Dark Mode
* Light Mode

Dark mode is the default.

---

## Code Quality

* Keep components modular
* Keep components reusable
* Avoid unnecessary complexity
* Use clean imports
* Use descriptive naming
